# Contributing

## Scope

This repository is a public baseline template for governance and technology documentation.

## Contribution Guidelines

- Keep content organization-agnostic in upstream.
- Do not include credentials, real tenant identifiers, or personal data.
- Do not include internal inventories (accounts, phone lists, incident evidence).
- Use placeholder values for examples.

## Document Standards

Policy and procedure documents should include:

- Version
- Effective date
- Next review date
- Document owner role

## Pull Requests

Pull requests should include:

- Summary of changes
- Reason for change
- Any backward-compatibility notes for downstream forks
