## Risk Rating Guidance

| Rating | Description |
| ------ | ----------- |
| High   | Likely to cause significant confidentiality, integrity, or availability impact |
| Medium | Material weakness with partial compensating controls |
| Low    | Limited impact, minor control weakness |

## Workflow

1. Identify the control gap.
2. Create a POA&M item with owner and due date.
3. Link to implementation and evidence references.
4. Validate remediation before closing item.
