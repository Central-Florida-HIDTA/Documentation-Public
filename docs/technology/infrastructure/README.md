# Infrastructure Documentation

## Overview

This directory contains platform documentation templates for common enterprise infrastructure domains.

## Included Templates

- `entra-id.md`
- `exchange-online.md`
- `microsoft-teams.md`
- `sharepoint-online.md`
- `intune.md`
- `conditional-access-policies.md`
- `security-compliance-center.md`
- `m365-licensing.md`
- `m365-api-access.md` (boundary note pointing tenant-specific auth and Graph inventory to the private Microsoft365 repo)

## Usage Guidance

- Keep public documentation at architecture and control level.
- Keep tenant inventories, account rosters, and operational evidence in private repositories.
- Use role-based ownership labels instead of personal identifiers.
